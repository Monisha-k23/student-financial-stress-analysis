# 📊 Student Financial Stress Analysis

## 📌 Project Overview
This project analyzes financial stress among students using EMI calculations and data analytics techniques.

## 📂 Dataset
A custom dataset was created including:
- Loan Amount
- Interest Rate
- Income
- Expenses
- Savings
- City Type

## ⚙️ Methodology
- Calculated EMI based on loan parameters
- Derived EMI-to-Income ratio
- Classified stress levels:
  - Low Stress
  - Medium Stress
  - High Stress

## 📊 Dashboard Features
- Stress Level Distribution
- City Type Filter (Rural, Tier 1, Tier 2)
- EMI-to-Income Ratio KPI
